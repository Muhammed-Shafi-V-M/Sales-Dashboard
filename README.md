## 👋 About the Project

This project is an interactive **Sales Dashboard** developed using **Power BI** to analyze sales performance, customer behavior, profit trends, and payment methods.

The dashboard transforms raw sales data into meaningful visual insights that help in understanding business performance and making data-driven decisions.

Using Power BI visualizations and interactive filters, users can explore the data dynamically and gain valuable insights quickly.

* Payment mode distribution
* Interactive Quarterly filters (Slicers)
